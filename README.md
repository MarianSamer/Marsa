# Elena Vasquez — Architecture Portfolio

A modern, responsive architectural portfolio website built with React, TypeScript, Vite, and Framer Motion.

## Features

- **Home Page**: Full-screen hero, project overview, image carousel with fullscreen view, design process timeline, and call-to-action
- **About Me Page**: Portrait header, biography, animated skills, contact form with validation
- **Animations**: Fade-in on scroll, parallax effects, staggered cards, page transitions, loading screen
- **Responsive**: Optimized for desktop, tablet, and mobile

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Customization

Edit `src/data/content.ts` to update:
- Architect name and bio
- Project details and images
- Skills and contact information
- Social media links

Replace images with your own architectural renders. Add your portfolio PDF to `public/portfolio.pdf`.

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18 + TypeScript
- Vite
- Framer Motion
- React Router
